## üì± Mobile Installation

Omni-Azul is a **Progressive Web App (PWA)**. For the best experience on iPhone:

1.  Open the site in **Safari**.
2.  Tap the **Share** icon (square with up arrow).
3.  Select **"Add to Home Screen."**
4.  The **Sphynx Icon** will appear on your grid. Opening it from the home screen removes the browser address bar for a native terminal experience.
